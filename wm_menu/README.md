# 世界地图菜单 VRAM 提取 (World Map Menu VRAM Extraction)

## 概述

本目录包含从 Metal Max Returns (MMR) 游戏中提取的菜单交互和配置界面 VRAM 数据。
通过在游戏探索阶段尝试不同按钮(X/Y/Select/Start), 发现了**Select按钮打开配置菜单**的重要机制。
共捕获 22 个独特的 VRAM 快照,记录了从游戏探索到配置菜单打开和导航的完整过程。

## 重大发现

### Select 按钮打开配置菜单

在帧 12400 按下 Select 按钮后, 游戏打开了**配置/设置菜单**, 包含以下选项:

| 配置选项 | 当前值 | 说明 |
|----------|--------|------|
| 战斗模式 (Battle Mode) | - | 战斗模式设置 |
| 信息速度 (Message Speed) | SLW | 消息显示速度 (Slow/Medium/Fast) |
| 行动方式 (Action Mode) | TURNS | 行动方式 (TURNS/ROUND) |
| 战斗动画 (Battle Animation) | ON | 战斗动画开关 |
| 战车遇阻时降下 (Vehicle Obstruction) | 全手动 | 战车遇阻行为 |
| 地名显示 (Location Name Display) | MENU/ENTER | 地名显示方式 |
| 声道设定 (Sound Channel) | STEREO | 声道设置 |

底部说明:
- **PRESS START : SAVE** - 在配置菜单中按Start保存
- **SELECT+START : RESET** - Select+Start组合键重置

### 按钮功能总结

| 按钮 | 功能 | 说明 |
|------|------|------|
| X | 调查/检视 | 显示"眼前并没有人…"对话框 |
| Select | 打开配置菜单 | 显示战斗/音效/显示等设置 |
| Start (配置菜单中) | 保存 | 在配置菜单中按Start保存游戏 |
| Select + Start | 重置 | 重置设置到默认值 |
| A | 确认/交互 | 在对话框中确认, 在菜单中选择 |
| B | 取消/关闭 | 关闭对话框或菜单 |

## 采集方法

- **工具**: Mesen2 2.1.1, Lua 脚本自动化
- **脚本**: `extract_wm_menu.lua`
- **运行帧数**: 0-14000 (约 233 秒游戏时间)

### 导航流程

| 阶段 | 帧范围 | 操作 | 说明 |
|------|--------|------|------|
| 跳过片头 | 3000-6800 | Start + A 连按 | 跳过标题画面和对话 |
| 关闭菜单 | 7000-7100 | B 键 | 关闭初始菜单 |
| 探索导航 | 7100-11000 | 右/上/右/上 | 在游戏区域内移动 |
| X键尝试 | 11000-11040 | X 键 | 尝试打开菜单 (触发检视) |
| 菜单导航 | 11100-12000 | 下/A/B 循环 | 在游戏场景中导航和交互 |
| Y键尝试 | 12200-12240 | Y 键 | 尝试Y键 (无明显效果) |
| **Select键** | 12400-12440 | **Select** | **打开配置菜单!** |
| 关闭尝试 | 12500-12540 | B 键 | 尝试关闭 (配置菜单保持打开) |
| 方向键导航 | 12600-14000 | 上/右/下/左 | 在配置菜单中导航选项 |

## VRAM 分析

所有 22 个捕获都有唯一的 VRAM 哈希值, 表明每个状态都有独特的 VRAM 数据:

| 场景 | 帧号 | VRAM Hash | 说明 |
|------|------|-----------|------|
| wm_00_before_menu | 11000 | b1b5da21a6e3 | 游戏探索 - 按X前 |
| wm_01_menu_open_x | 11100 | 56f70d93e6fe | X键后 - 显示"眼前并没有人…" |
| wm_02-07_nav_down | 11180-11580 | 各不相同 | 导航和交互过程 |
| wm_08_submenu_open | 11620 | 478c730c3194 | A键 - 对话框 |
| wm_09-14 | 11700-12100 | 各不相同 | 关闭和返回过程 |
| wm_15-17 | 12200-12400 | 各不相同 | Y键和Select键尝试 |
| **wm_18_after_select** | **12500** | **59eb7e9ea2ee** | **配置菜单打开!** |
| wm_19_walking_start | 12600 | 7be31ff30356 | 方向键导航配置菜单 |
| wm_20_terrain_1 | 13100 | f85c288ab42e | 配置菜单 - 光标在信息速度 |
| wm_21_terrain_2 | 13600 | af4c62232101 | 配置菜单 - 继续导航 |

## 技术说明

- 游戏在帧 11000 时仍处于室内/地下区域, 未到达世界地图
- X 按钮触发"检视"动作 (显示"眼前并没有人…"), 不是菜单按钮
- Select 按钮是打开配置菜单的正确按钮
- 配置菜单中的方向键操作会导航菜单选项, 而非移动角色
- 配置菜单中的 "PRESS START : SAVE" 表明 Start 按钮在配置菜单中用于保存
