# ROM 结构分析 (ROM Structure Analysis)

## 概述

本目录包含 Metal Max Returns ROM 文件的结构分析结果。
通过计算每个 8KB bank 的 Shannon 熵和零字节比例，区分了代码/数据/空 bank。

## ROM 基本信息

| 属性 | 值 |
|------|-----|
| ROM 大小 | 4,194,816 字节 (4.00 MB) |
| SMC 头部 | 无 (512 字节偏移, 0x00-0x07 全零但 0x08-0x0F 非零) |
| 映射模式 | LoROM ($30) |
| ROM 类型 | $02 (FastROM) |
| SRAM | 8 KB (带电池) |
| 游戏代码 | AZMJ |
| 开发系统 | KUSOU KAGAKU DEVELOPMENT SYSTEM VER 2.19 |

## 中断向量

| 向量 | 地址 |
|------|------|
| NMI | $00:DE65 |
| RESET | $00:0090 |
| IRQ | $DE5B |

## Bank 分类统计

| 类型 | 数量 | 大小 | 说明 |
|------|------|------|------|
| CODE/TBL (代码/表) | 94 | 3008 KB | 低熵, 可能为代码或查找表 |
| DATA (数据) | 34 | 1088 KB | 高熵 (>7.0), 图形/音频/文本数据 |
| EMPTY (空) | 0 | 0 KB | 无完全空的 bank |

## 数据 Bank 分布

| Bank | ROM 地址 | 熵 | 推测内容 |
|------|----------|-----|---------|
| $09, $0A | 0x048000-0x05FFFF | 7.3-7.4 | 图形数据 |
| $12 | 0x090000-0x097FFF | 7.0 | 地图/脚本数据 |
| $1B-1D | 0x0D8000-0x0FFFFF | 7.1-7.2 | 音频/文本数据 |
| $20, $22-25 | 0x100000-0x12FFFF | 7.0-7.4 | 图形数据 |
| $2A, $2C, $2E, $30-32 | 0x150000-0x19FFFF | 7.0-7.3 | 图形/音频 |
| $34 | 0x1A0000-0x1A7FFF | 7.4 | 高熵数据 |
| $40, $42-4F | 0x200000-0x27FFFF | 7.1-7.3 | 大规模数据区 |
| $52-5B | 0x290000-0x2DFFFF | 7.0-7.3 | 扩展数据 |

## ROM 区域密度

| 地址范围 | 熵 | 说明 |
|----------|-----|------|
| 0x000000-0x00FFFF | 5.1 | 代码/向量/开发系统标识 |
| 0x010000-0x07FFFF | 6.1-6.8 | 主代码区 |
| 0x080000-0x0FFFFF | 6.2-7.3 | 代码+数据混合 |
| **0x0E0000-0x2C0000** | **7.0-7.5** | **连续高熵数据区 (1.9 MB)** |
| 0x2D0000-0x2FFFFF | 1.7-6.8 | 代码+填充区 |
| 0x300000-0x39FFFF | 2.7-6.7 | 代码+压缩数据 |
| **0x3A0000-0x3BFFFF** | **0.0** | **空 bank (128 KB 全零/全FF)** |
| 0x3C0000-0x3FFFFF | 2.7-6.2 | ROM 尾部 |

## 关键发现

1. **数据集中在 0x0E0000-0x2C0000**: 约 1.9MB 的连续高熵区域，包含游戏主要数据资产（图形、音频、地图数据）。
2. **空 bank 0x3A0000-0x3BFFFF**: 128KB 完全未使用（全 0xFF），可能是 ROM 空间未填满。
3. **SPC700 音频固件**: 位于 ROM 开头 0x000000 区域。
4. **开发系统**: "KUSOU KAGAKU DEVELOPMENT SYSTEM VER 2.19" - 空想科学开发系统 2.19 版。

## 技术说明

- LoROM 映射: Bank $XX 中地址 $8000-$FFFF 映射到 ROM 偏移 $XX × $8000 + (addr & $7FFF)
- Bank $40-$7F 是 $00-$3F 的镜像 (LoROM)
- Bank $80-$BF 和 $C0-$FF 也是 $00-$3F 的镜像
- 因此 4MB ROM 实际使用 128 个唯一 bank (64 个原始 × 2 镜像)
