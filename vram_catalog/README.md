# 跨会话 VRAM 综合分析与图块目录 (Comprehensive VRAM Catalog)

## 概述

本目录包含对所有提取会话中 VRAM 数据的综合分析结果。分析涵盖了 7 个提取目录中的 105 个 VRAM 转储, 对图块数据进行去重和编目, 生成了主图块表和调色板比较图。

## 分析结果摘要

| 指标 | 数值 |
|------|------|
| VRAM 转储总数 | 105 |
| 唯一 VRAM 状态 | 105 (0% 重复) |
| 唯一 CGRAM 调色板 | 2 |
| 静态字符区非空图块 | 588 / 1024 (57.4%) |
| 动态字符区非空图块 | 390 (跨场景累计) |

## VRAM 区域分析

| 区域 | 地址范围 | 唯一模式数 | 说明 |
|------|----------|------------|------|
| BG1 Tilemap | 0x0000-0x07FF | 105 | 每个转储都不同 (玩家位置变化) |
| BG2 Tilemap | 0x0800-0x0FFF | 77 | 部分场景共享 BG2 布局 |
| BG3 Tilemap | 0x1000-0x17FF | 100 | 大部分转储不同 |
| 未使用区域 | 0x1800-0x1FFF | - | 全零 |
| 空区域1 | 0x2000-0x3FFF | - | 全零 |
| 动态字符区 | 0x4000-0x5FFF | 101 | 场景相关图块图形 |
| 空区域2 | 0x6000-0x7FFF | - | 全零 |
| **静态字符区** | 0x8000-0xFFFF | **1** | **所有 105 个转储完全相同** |

### 关键发现

1. **静态字符区完全一致**: 所有 105 个 VRAM 转储的 0x8000-0xFFFF 区域完全相同, 确认为全局共享图块图形。
2. **零重复率**: 所有 105 个 VRAM 转储都是唯一的, 表明每次捕获都捕捉到了不同的游戏状态。
3. **仅 2 个调色板**: 所有 105 个转储只使用了 2 个 CGRAM 调色板, 表明游戏在大部分场景中使用相同的颜色方案。
4. **BG1 每次都不同**: BG1 tilemap 在每个转储中都不同, 因为它直接反映玩家在地图上的位置。

## 图块格式

- **格式**: SNES 4bpp (每像素 4 位, 16 色)
- **图块大小**: 8×8 像素, 32 字节
- **编码**: 4 个位平面, 每 2 字节为一对 (位平面 0&1, 位平面 2&3)
- **调色板**: 从 CGRAM 读取, BGR555 格式
