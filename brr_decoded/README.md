# BRR音频样本深度解码

## 概述

通过DSP寄存器DIR($5D)定位SPC700 RAM中的样本目录，解码BRR(Bit Rate Reduction)压缩音频样本为WAV文件。

## 解码结果

| 指标 | 值 |
|------|-----|
| SPC转储数 | 12 |
| 解码样本总数 | 86 |
| 唯一样本指针 | 23 |
| WAV文件数 | 86 |
| 采样率 | 32,000 Hz (SPC700原生) |
| 格式 | 16位 PCM WAV (单声道) |

## DSP寄存器分析

所有场景的DSP寄存器状态：

| 寄存器 | 值 | 说明 |
|--------|-----|------|
| DIR ($5D) | 0x04 | 样本目录在SPC RAM 0x0400 |
| FLG ($6C) | 0x00 | DSP正常工作(未静音/重置) |
| ESA ($5E) | 0x00 | 回声缓冲区在0x0000 |
| KON_L ($4C) | 场景相关 | Key On低位 |

### 各场景KON状态

| 场景 | KON | 活跃Voice | 说明 |
|------|-----|-----------|------|
| title | 0x87 | 0,1,2,7 | 标题音乐4声道 |
| intro_end | 0x7E | 1,2,3,4,5,6 | 6声道活跃 |
| town | 0xA0 | 5,7 | 城镇2声道 |
| wm_entry | 0xA0 | 5,7 | 世界地图2声道 |
| config_open | 0x7E | 1-6 | 配置菜单6声道 |

## BRR解码算法

### BRR块格式 (9字节)
- **字节0 (头)**: bits 4-7=范围移位, bits 2-3=滤波器类型, bits 0-1=结束/循环标志
- **字节1-8 (数据)**: 16个4位样本 (高4位+低4位交替)

### 滤波器类型
| Filter | 公式 |
|--------|------|
| 0 | 无滤波 |
| 1 | sample += old1 + (-old1 >> 4) |
| 2 | sample += 2*old1 - (3*old1 >> 5) - old2 + (old2 >> 4) |
| 3 | sample += 2*old1 - (7*old1 >> 6) - old2 + (13*old2 >> 6) |

### 结束标志
- 0: 继续
- 1: 循环点
- 3: 结束
